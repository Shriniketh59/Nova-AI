# Project Memory

## Nova AI Research Platform Frontend
A modern research hub interface for AI-assisted peer review, semantic literature search, and system configurations.

### Tech Stack
- **Framework**: Vite + React
- **Routing**: React Router (`react-router-dom`)
- **Styling**: Tailwind CSS v4 (integrated via `@tailwindcss/vite` plugin)

### Core Folder Structure
- `src/layouts`: Handles app frames and shared view wrappers.
- `src/pages`: Individual route components (Dashboard, Research, PeerReview, Settings).
- `src/components`: Reusable UI elements (GlassCard, StatsCard, Sidebar, Header).
- `src/services`: Utility APIs and mock data providers (api.js).
